# Мобильное приложение для создания и редактирования игровых персонажей

Данный проект разработан в рамках производственной практики.

## Описание
Приложение позволяет создавать, редактировать и удалять игровых персонажей, а также сохранять и загружать их из файла.

## Функции
- Добавление персонажа
- Редактирование характеристик
- Удаление персонажей
- Сохранение и загрузка данных

## Технологии
- Python
- JSON
- Основы геймдизайна
